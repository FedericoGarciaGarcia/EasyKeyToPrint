# EasyKeyToPrint


## How to use

Compile with:

```
javac -cp ".;libs/jnativehook-2.1.0.jar" *.java

```

Run with:

```
java -cp ".;libs/jnativehook-2.1.0.jar" EasyKeyToPrint

```
